深度学习_使用autoencoder自动提取特征

发表于 2017-04-27   |   分类于 认知神经科学   |     |   阅读次数

autoencoder自动提取特征的原理是使用输入数据作为输出标签训练数据，中间层的神经元的数目小于特征数目，使用中间层的神经元的输出作为特征，这样就达成一个特征空间变换和降维的目的。本质上它是通过一种非线性的变换函数在转换特征空间。
